## Instalación

Para instalar las dependencias del proyecto, ejecuta:

```bash
npm install
```

## Uso

Para iniciar el servidor de desarrollo, ejecuta:

```bash
npm run dev
```

## Base de Datos

El proyecto utiliza una base de datos en formato JSON que se guarda en la carpeta public.
